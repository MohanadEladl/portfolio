# AI Engineer Portfolio – Mohanad ElAdl

A high-end, dark-mode professional portfolio targeting HR recruiters and tech collaborators. Built with a focus on AI aesthetics, advanced motion, and visual excellence.

## 🚀 Version 3.0 Updates
- **Neon Wordmark Logo**: Replaced the "M." token with a full "Mohanad ElAdl" wordmark with a neon blue glow.
- **Advanced Animations**: Added floating orbs, a neuron particle network, and a neon cursor trail.
- **Enhanced Project Cards**: Each card now features a high-quality screenshot with a neon frame and a tech-tag hover overlay.
- **Vibrant UI**: Neon text-shadows on all headings and animated glowing underlines on nav links.

## 🛠️ Setup & Deployment
1.  Clone the repository.
2.  Open `index.html` in a web browser to view locally.
3.  Deploy to GitHub Pages by pushing to the `main` branch.

> [!IMPORTANT]
> **Download Résumé Button**: Upload your `resume.pdf` to the repository root for the "Download Résumé" button to function correctly.

## 📁 Project Structure
- `index.html`: Core structure and content.
- `styles.css`: Advanced styling, animations, and responsive design.
- `script.js`: Interactive logic, particle systems, and cursor effects.
- `images/`: High-resolution project screenshots.
